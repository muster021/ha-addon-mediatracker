# Changelog

All notable changes to this add-on will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.2.11] - 2026-04-06

### Added
- Initial release of MediaTracker as a Home Assistant add-on
- Home Assistant calendar integration via iCal feed (`/api/calendar.ics`)
- Sonarr integration: import monitored series to watchlist
- Discover page: trending TV shows from TMDB
- Streaming availability via TMDB watch providers
- HTTP Basic Auth support for iCal feed (compatible with HA Remote Calendar)
- Support for amd64 and aarch64 architectures
